# Responsive Login Form

A simple and modern login form created with HTML and CSS. This project is fully responsive and features a clean interface with custom fonts and icons for email and password fields.

## Key Features

  * **Responsive Design:** The layout adapts seamlessly to different screen sizes, from mobile devices to desktops.
  * **Modern UI:** A clean, card-based interface for a modern look.
  * **Standard Inputs:** Includes fields for E-Mail and Password.
  * **User-Friendly:** Features a "Keep Me Logged In" checkbox and a "Forgot Password" link.
  * **Pure CSS:** Styled entirely with CSS, with no JavaScript dependencies.

## Installation and Usage

To use this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/dusfirefox/responsive-login-form.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd responsive-login-form
    ```
3.  **Open the file:**
    Simply open the `index.html` file in your web browser to view the login page.

## Technologies Used

  * **HTML5:** For the structure of the login form.
  * **CSS3:** For all styling, layout, and responsiveness (including media queries).
